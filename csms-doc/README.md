csms readme
===================

fdu-csms为复旦大学数据库课程大作业--高校选课管理系统。

主要功能有：

* 教务员

	*基本信息维护
	
	*开设课程
	
	*修改课程
	
	*查看所有课程
	
	*查看学生的选课情况

* 教师

	*基本信息维护
	
	*开设课程
	
	*查看选课名单
	
	*提交成绩

* 学生

	*基本信息维护
	
	*选课
	
	*退课
	
	*查看所选课程
	
	*查看成绩
	
	*评教
